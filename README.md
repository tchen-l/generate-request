# generate-request README

语忆 - 用于前端自动生成接口代码

## Features

配置完接口文档登录信息后，运行此扩展，输入对应接口 id 后自动生成接口代码

## Extension Settings

扩展设置项:

* `generateRequest.apiHost`: http://localhost:5000
* `generateRequest.username`: guest
* `generateRequest.password`: 123456
